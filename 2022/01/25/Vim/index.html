Vim 哲学

Vim 是最流行的command-line-based editor，VSCode上服务器有点重。

  • Vim是可编程的，它的interface是一种编程语言，Composable
  • Vim避免使用鼠标
  • Vim避免使用方向键

三个主要模式

默认左下角会显示当前所处的模式（无显示即为Normal Mode）

  • Normal Mode (Command Mode)
  • Insert Mode: 插入文本
  • Visual Mode: 选择文本

基础的是通过 : i / v 从Normal Mode切换到Insert/Visual Mode; 而后通过ESC切换回Normal Mode。

从Normal Mode 切换到 Insert Mode
  • 在当前光标前插入：i
  • 在当前光标后插入：a
  • 在行首插入：I
  • 在行尾插入：A
  • 在下方插入一行：o
  • 在上方插入一行：O

作为编辑器的一些操作

光标移动
  • 基础移动
    • 上下左右：hjkl -- 左下上右
    • 移动到第一行：gg
    • 移动到最后一行：G
    • 移动到第x行：xG
  • 进阶移动
    • 移动到下一个单词的开头：w
    • 移动到下一个单词的结尾：e
    • 移动到上一个单词的开头：b
    • 移动到行首：0
    • 移动到第一个非空字符：^
    • 移动到行尾：$
    • 移动到变量定义处：gd
    • 移动到前一个没有匹配的左大括号出：[{
复制粘贴

  • 复制：y

    • 复制整行：yy
    • 复制当前行开始的n行：nyy

  • 粘贴：p

    • 在当前光标下一行粘贴：p
    • 在当前光标上一行粘贴：P
Undo/Redo
  • Undo：u
  • Redo：Ctrl+r
删除(实际上是剪切)
  • 删除整行：dd
  • 删除字符：x
  • 删除到行尾D
替换
  • 替换字符：r
  • 替换到行尾：C/Da
  • 修改大小写~
可组合性(Composable)

数字加指令表示将这个指令执行n次

  • 删除两个单词：d2w
  • 删除单词做两遍：2dw
  • 前两者合起来：2d2w
搜索替换*
  • 搜索下一个xxx：/xxx
  • 搜索上一个xxx：?xxx
  • 在/xxx或者?xxx后继续按照这个方向检索下一个：n
  • 在/xxx或者?xxx后反方向检索下一个：N
  • 快速检索当前光标处的单词：#
  • 将range范围内的from替换为to，同时注意flag：:[range]s/from/to/[flag]

Split Window

分出多个窗口来看一个文件，这些窗口基于同一个buffer，修改是互通的。

  • sp上下分窗
  • vsp左右分窗
  • 在窗口间移动：Ctrl+w +[hjkl]
Mark

标记代码的一个地方用于后续快速跳转到此处

  • 生成一个Mark：m
  • 跳转到一个Mark处 `
