Communication: Pipe & Redirect
  • Pipe: Command_1 | Command_2 :把Command_1的stdout作为Command_2的stdin。
  • Redirect：
    • Command_1 > file : Command_1的stdout输出到file中
    • Command_2 < file : file内容作为Command_2的stdin
    • 0: stdin, 1: stdout, 2:stderr
