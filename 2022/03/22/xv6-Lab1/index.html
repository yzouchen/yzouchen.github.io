xv6 Lab1 Utilities

发表于 2022-03-22 分类于 Labs ， xv6 阅读次数：
本文字数： 6.9k

官方实验指导书：Lab: Xv6 and Unix utilities (mit.edu)

本实验要求实现Unix中几个常用的函数：sleep, pingpong, primes, find, xargs

sleep

在user/下创建sleep.c文件，根据指导书，调用系统调用中的sleep()实现，这就很简单了啊，主体就是系统调用的sleep()，在此基础上，增加一些判断参数个数（不写也不影响自动判分的正确性），调用atoi()将参数转化为int型，判断参数大小，调用就完事了。

pingpong

这一部分主要是为了熟悉OS中父子进程的关系、通讯，以及管道的使用。

具体来说是：父进程发送ping，子进程收到后发送pong，收到后均打印出来。

根据HINTS也可以得到：

  • Use pipe to create a pipe.
  • Use fork to create a child.
  • Use read to read from the pipe, and write to write to the pipe.

注意 READ对应管道端为 0 号端口， WRITE为 1 号端口

primes

​ 本部分利用pingpong中提到的fork()和pipe()实现质数筛选的功能

​ 具体来说就是，每个子进程从父进程那接受现在不认为是素数的数，此时这些数中的最小数（第一个数）不能被比其小的数 整除，可以认为是素数，那么将那些数往后传呢？这就要进行过滤，将当前判断的第一个数的整数倍数，均过滤掉。即，每次过滤掉当前最小数（质数）的倍数。

xargs

这一小部分主要是为了帮助理解UNIX操作系统中，执行一个程序/Command的过程。

即通过fork+exec系统调用实现。

xargs题意是，存储下xargs命令后面跟的参数，将其与后面新的参数表结合在一起，形成新的（指令+）参数表。由于指令也会放在argv[0]，所以就是全部拼在一起就是了。
